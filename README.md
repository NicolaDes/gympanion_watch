# Garmin GymPanion Watch App

A garmin watch application to track and monitor gym excercises.

## Directory structure

```
.
├── app                                := Garmin application
│   ├── resources                      := Garmin application resources
│   │   ├── drawables                  := Garmin application drawables
│   │   ├── layouts                    := Garmin application layouts
│   │   ├── menus                      := Garmin application menus
│   │   └── strings                    := Garmin application strings
│   └── source                         := Gramin application monkey c source code
└── scripts                            := Scripts used outside of Garmin application
```
